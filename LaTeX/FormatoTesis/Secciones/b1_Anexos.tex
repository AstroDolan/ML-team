\section{Segundo anexo}%
\label{anex:anexo2}%
Algunos ejemplos de anexos:%
\begin{itemize}
	\item documentos legales o normativos,%
	\item manuales de equipos usados,%
	\item artículos o estándares técnicos (ej.: ISO).%
\end{itemize}%

Los anexos no son críticos para entender el trabajo. Se incluyen como evidencia o para cumplir requisitos formales.\par%
