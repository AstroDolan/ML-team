\begin{abstract}
    Se realizó un acople entre una Physics-Informed Neural Network y una red neuronal bayesiana para la predicción de la morfodinámica de un lahar de un volcán arbitrario del sur de Chile.   
\end{abstract}
\keywords{Physics-Informed Neural Network, Bayesian Neural Architectures, Partial diferential equations}

\section{Objetivos de la Tesis}
\subsection{Objetivo General}
Se crea una red neuronal informada por la física acoplada con una red neuronal bayesiana para resolver un sistema de 6 ecuaciones diferenciales parciales que predicen la morfodinamica de un lahar en un volcan activo del sur de Chile
\subsection{Objetivos Específicos}
\begin{enumerate}
    \item Se debe formar un sistema de ecuaciones compuesto con, por lo bajo, 6 ecuaciones diferenciales parciales que modelen la morfodinámica de un lahar arbitrario de un volcán de Chile.
    \item Se construye una red neuronal informada por la física usando el sistema de PDEs y otros parámetros geológicos que logre predecir rigurosamente el comportamiento real de un flujo de sedimento. 
    \item Se acopla una red neuronal bayesiana a la PINN de manera que los pesos aleatorios de la nueva red usen distribuciones de probabilidad y se encuentre la distribución posterior usando métodos de Monte Carlo Hamiltoniano o inferencia variacional para reducir los errores físicos.
    \item Se debe lograr predecir la morfodinámica de manera extremadamente realista y acorde a la física para poder exportar un mapa de peligro volcánico que entregue datos de evacuación según el riesgo calculado por la red neuronal.   
\end{enumerate}
